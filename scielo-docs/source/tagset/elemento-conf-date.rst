.. _elemento-conf-date:

<conf-date>
===========

Aparece em:

  :ref:`elemento-element-citation`

Ocorre:

  Zero ou mais vezes

Identifica a data de uma conferência, evento etc. Pode ser também um período, por exemplo: 2003 Aug 25-29.

Exemplo:

.. code-block:: xml

    ...
    <element-citation>
        <conf-date>2002 Jul 28-Aug 2</conf-date>
    </element-citation>
    ...
