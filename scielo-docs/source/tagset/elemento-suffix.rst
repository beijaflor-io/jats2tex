.. _elemento-suffix:

<suffix>
========

Aparece em:

  :ref:`elemento-name`

Ocorre:

  Zero ou uma vez


Especifica sufixos de nome de autor, como por exemplo, "Neto", "Júnior", "Jr.", "Filho", "Sobrinho" etc.

Exemplo:

.. code-block:: xml

    ...
    <name>
        <surname>Santos</surname>
        <given-names>João da Silva</given-names>
        <suffix>Neto</suffix>
        ...
    </name>
    ...
