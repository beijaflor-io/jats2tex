.. _elemento-copyright-year:

<copyright-year>
================

Aparece em:

  :ref:`elemento-permissions`

Ocorre:

  Zero ou mais


Identifica o ano do direito autoral com quatro dígitos e sem espaços.

Exemplo:

.. code-block:: xml

    ...
     <article-meta>
       ...
        <permisssions>
           ...
            <copyright-year>2014</copyright-holder>
           ...
        </permissions>
        ...
     </article-meta>
